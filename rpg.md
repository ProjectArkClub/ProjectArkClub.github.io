## 写在前面
本程序支持自动后台变速、找怪、捡物、皇家寻路、转盘、扭蛋、兑换材料，支持自动对战包括上卡、主副宠进化、捕捉、召唤、攻击，战后支持补卡、补evp、补buff、副宠复活、过答题、断线重连等功能，支持全服

初次使用请阅读 <A href="/setup">基础设置说明</A>

## RPG 界面
> 所有功能均支持后台自动运行

  ![12]({{site.cdn}}/resource/rpg.png)

- 功能提示：鼠标在图标上即可触发提示

  ![12]({{site.cdn}}/resource/tooltips.png)

- 转盘: 自动打开转盘抽奖
- 野怪: 
  - 选择自动识别的野怪名字或自动寻路模式
  - "#Maze-Random-Run" 为迷宫随机跑模式，注意事先将人物移动至远离传送门，似乎不支持单人迷宫
  - “自动RB”系列为自动寻路，可从关键路径中任意点开始自动寻路至该房终点
  - **注意**：
    - 野怪名字选错则**不选怪**
    - 私服勾选已重置的野怪（名字后缀为-re）识别效率更高
    - 若野怪被人物、数码兽、各种光影效果、地形遮挡，则无法识别
    - 建议屏蔽其他用户模型，携带模型较小、特效较少的数码兽，佩戴特效较少的饰品，选择地形遮挡较少、人少的点位
- 找怪|寻路: 
  - “开启”则原地自动找怪发起对战，手刷列表外数码兽时设为“关闭”可提升对战识别速度
  - 寻路中切至“关闭”可暂停寻路；切至“开启”则继续寻路
- 上卡: 自动刷卡，默认用背包卡槽123的卡
  - 关闭: 不上卡
  - 开启: 自动上卡
  - 自动: 自动上卡并补卡，用完后从**主背包**自动将相同的卡片插入背包的战斗卡槽中。支持中途手动换卡，不需要暂停挂机
  - 极速：智能上卡并补卡，需要开启 RPG 自带的自动上卡（如公益服/火神服），速度最快
- 召唤: 
  - 单招: 队长召唤一只宠
  - 高速气场单招：若高速BB死亡则可复活
  - 轮流双招: 队长召唤副宠、副宠召唤下一只宠
  - 连续双招: 队长连招两只宠
  - 宠槽：副宠编号与L键编号对应，高速BB是1号
    - 槽1: 第一次召唤槽或单招
    - 槽2: 第二次召唤槽
  - 复活：
    - 召唤时判定副宠生死状态，若死亡则**战斗后**使用主背包复活药复活；
    - 仅“高速气场单招”模式可主动检测高速BB死亡状态
- 进化: 队长自动进化
- 捕捉: 
  - 1hp 模式要求将一血卡放至卡槽1，且上卡调为关闭
- 对战
  - 最大怪物数：
    - 0 或“打一次”模式在对战中**不打怪**
    - 1~5 自动对战，均具备抢先手能力，区别在于自动出手次数
  - 技能: 队长攻击技能，实时生效
- evp: 主兽 evp < 25% 吃药
  - 下拉菜单选择药品种类
  - cfg.json 设置单次吃药个数
    
        ...
        "RPG":{
          ...
          "evp":{
              "type": 1,
              "times": 4 （代表一次吃4瓶evp药）
          }
          ...
        }
- 捡物: 北美服背包满后不再拾取
  - 自定义捡物：
    - UI 界面选择“多捡”
    - 右键任务栏小图标 - RPG捡物 - 勾选待拾取道具

      ![13]({{site.cdn}}/resource/select_item.png)
      ![13]({{site.cdn}}/resource/select_item_2.png)
  - 单捡：选择 UI 下拉菜单中的一项道具

      ![13]({{site.cdn}}/resource/select_item_3.png)
  - 不捡：关闭
  - 注意：
    - 已知在机器**CPU高负载**情况下易出现捡物不灵的现象，建议**关闭其他CPU负载较高程序**缓解这一现象
    - 不同规格显示器显示像素有差异，出现列表中已勾选物品无法拾取的情况请咨询作者并告知显示器参数
- 兑换:
  - 道具标号：兑换聊天框中目标道具的标号
  - 兑换次数：兑换几次
  - 用法：设置好后点运行，然后手动与NPC对话即可
- 变速优化: 进一步提升变速效果，影响键盘输入
- 保存配置：点击 save 按钮或组合键 ctrl-s

## RCONFIG 界面

  ![12]({{site.cdn}}/resource/rpg_config.png)

- 自动登录：仅公益服/火神服/服不服断线自动重连
  - 填写账号、密码、角色、登录器
  - 登录器：一般是 DigimonRPG.exe
  - 注意：登录界面**不要勾选“记住ID”**
- 变速：官服自动禁用
  - 野外变速：手操跑图建议 1.6 倍，挂机时随意
  - 战斗变速：10 倍以上差距不大
  - 组队时建议和队友同步倍率
- 答题：
  - 关闭：人工答题，方便手操
  - 开启：100% 准确自动答题
- 扭蛋：选择自动，回到 RPG 页面点击“运行”再手动打开扭蛋机即可
- RPG 界面设置：
  - 阴影：开启或关闭阴影
  - 上卡：开启或关闭自动上卡
  - 技能：开启或关闭技能动画
  - 地图：开启或关闭小地图
  - 聊天：聊天框小中大
  - 默认模式则程序不自行设置
